# ServiceNow IT Help Desk System

A complete IT Help Desk system built on ServiceNow PDI.

## What This Project Does
This project builds a fully working IT Help Desk system
on ServiceNow from scratch including service catalog,
incident management, automation, notifications,
reports and knowledge base.

---

## Project Phases

### Phase 1 - Service Catalog
- Created IT Services category
- Built 3 catalog items:
  - Request a Laptop
  - VPN Access Request
  - Software Installation
- Added variables - dropdowns, checkboxes, date fields
- Written client script to show/hide fields dynamically

### Phase 2 - Incident Management and Flow Designer
- Created Flow Designer flow to auto create incidents
- Set trigger on catalog request submission
- Mapped all incident fields automatically

### Phase 3 - Business Rules and Scripts
- Created 4 support groups:
  - Software Support
  - Hardware Support
  - Network Team
  - IT Help Desk
- Written Business Rule to auto assign incidents to correct team
- Written Client Script to show resolution fields when resolved

### Phase 4 - Email Notifications
- Incident Assigned - notifies assignee
- Incident Resolved - notifies caller
- Incident Updated - notifies caller on comments

### Phase 5 - Reports and Dashboard
- Open vs Closed Incidents pie chart
- Incidents by Category bar chart
- SLA Breach Rate bar chart
- IT Help Desk Overview dashboard

### Phase 6 - Knowledge Base
- Created IT Help Desk KB
- Published 3 KB articles:
  - How to connect to VPN
  - Laptop request process and timelines
  - How to request software installation
